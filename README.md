# Dialog Elderly

## Overview
This repository contains various mathematical formulas applicable in the context of elderly interaction.

### Mathematical Formulas
- The equation of a line: $y = mx + b$ \
- The quadratic formula: $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$ \
- The area of a circle: $A = \pi r^2$ \
- E=mc² (Energy-Mass equivalence) \

## Usage

Ensure to escape underscores (_), for example: $A_{x} = \sqrt{ \sum_{i=1}^{n} x_{i}^{2} }$
